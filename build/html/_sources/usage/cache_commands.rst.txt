
Setup Cache Commands
####################

Need to config mail in Dashboard -> Cache Commands.

Clear Web caching: database caching, static blocks...
Run this command when you don't see the changes after updating data.

.. image:: /images/cache_commands.png

Using Cache Commands
====================

Example:

* Click ``Clear all Web cache``
    * Clear Web caching: database caching, static blocks... Run this command when you don't see the changes after updating data.
* Click ``Refresh compiled views``
    * Clear compiled views to make views up to date.
* Click ``Clear config cache``
    * You might need to refresh the config caching when you change something on production environment.
* Click ``Clear route cache``
    * Clear cache routing.
* Click ``Clear log``
    * Clear system log files
